# ATE-Junbot

## Hardware

- Jetson Orin Nano
- RPLidar A2
- T265

## Feature 

- 4-wheel mecanum kinematic
- Build 2D map with lidar (2D Lidar + VIO from T265)
- Navigation in 2D map
- Obstacle avoidance
- Object detection
